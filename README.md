# Safe-Call (세이프콜)

> 보이스피싱 피해방지를 위한 통화 중 실시간 AI 조언 및 안심케어 서비스

## 📌 Overview

Safe-Call은 보이스피싱 범죄의 핵심 전술인 '피해자 고립'을 무력화하기 위해 설계된 AI 기반 실시간 보호 서비스입니다. 통화 중 사용자 옆에서 함께 듣고 조언하는 AI 컴패니언(Companion)을 통해 범죄를 선제적으로 차단합니다.

## 🎯 Core Features

### 1. 실시간 위험 탐지 및 경고
- **음성 분석**: STT를 통한 실시간 대화 텍스트화
- **위험도 시각화**: 기관 사칭, 협박, 금전 요구 등 핵심 패턴 탐지 후 색상 코드로 즉각 경고
- **대응 가이드**: 상황별 구체적인 대응 스크립트 실시간 제공

### 2. 실시간 사실 확인(Fact-Checking)
- 통화 중 언급되는 기관명, 담당자 이름, 사건번호 등을 자동 인식
- '대검찰청 찐센터' 등 공식 채널 DB와 연동하여 즉시 진위 확인
- OCR 모델을 통한 위조 문서 분석

### 3. 통화 후 안심 케어
- **개인정보 노출 리포트**: 통화 중 노출된 정보(계좌번호, 비밀번호 등) 명확히 안내
- **원클릭 신고**: 자동 정형화된 신고서를 관계 기관에 즉시 전송
- **통화 내용 정규화**: 통화정보, 피싱 분류, 상황 요약 자동 생성

### 4. 기관용 통합 대시보드
- 비식별화된 신고 데이터 실시간 집계
- 신종 수법, 지역·시간대별 통계 제공
- AI 피드백 루프를 통한 탐지 정확도 지속적 개선

## 🛠 Technical Architecture

### AI Models & Technologies
- **Speech Recognition**: Whisper.cpp 기반 실시간 음성 전사
- **Pattern Detection**: LLM 기반 보이스피싱 시나리오 패턴 분석
- **Document Analysis**: OCR 모델을 통한 문서 진위 판단
- **Data Integration**: 공식 DB 연동 API

### Core Workflow

```
[통화 시작] 
    ↓
[실시간 STT + LLM 분석]
    ↓
[위험도 판단 및 경고 표시]
    ↓
[사실 확인 및 대응 가이드 제공]
    ↓
[통화 종료]
    ↓
[개인정보 노출 분석 + 신고서 생성]
    ↓
[원클릭 신고 → 기관 대시보드 연동]
    ↓
[신종 패턴 학습 → AI 업데이트]
```

## 💡 Why Safe-Call?

### Problem
보이스피싱 범죄의 가장 큰 성공 요인은 피해자를 제3자로부터 철저히 단절시켜 정상적인 판단을 마비시키는 것입니다. 고립된 환경에서는 외부 조언이나 사실 확인이 불가능해 피해가 커질 수밖에 없습니다.

### Solution
AI를 통해 통화 중인 사용자 바로 옆에서 함께 듣고 조언해주는 동반자를 제공합니다. 실시간 사실 확인으로 범죄자의 거짓말을 즉시 탄로시키고, 심리적 압박 속에서도 올바른 판단을 내릴 수 있도록 지원합니다.

## 📊 Expected Impact

### 사회적 효과
- 범죄 피해 예방을 넘어 국민 개개인에게 심리적 안정감과 통제감 부여
- 범죄에 대한 사회 전반의 불안감 완화

### 경제적 효과
- 보이스피싱 직접 피해액 감소
- 피해 수습 과정의 행정적·사회적 비용 절감

### 행정·수사 효율
- 실시간 데이터 기반 신종 범죄 수법 신속 파악
- 데이터 기반 선제적 예방 정책 수립 가능

## 🔄 Continuous Improvement

새롭게 탐지된 범죄 패턴은 즉시 프롬프트에 업데이트되어 탐지 정확도를 높이고, 모든 사용자에게 최신 대응 조언으로 자동 업데이트되는 선순환 구조를 구축합니다.

---

**Safe-Call**은 단순한 탐지 서비스가 아닌, 사용자 곁에서 함께하는 AI 조언자로서 보이스피싱 범죄를 원천 차단하는 것을 목표로 합니다.
